Intro
=====

Alfred 2 workflow: 根据ip-api.com来查询 IP 地址信息

IP 地址库: http://ip-api.com/

Usage
=====

Alfred 框中输入 lip, 参数可以为:

    lip  # 查询本机的 IP 地址
    lip 199.91.73.222 178.79.131.110  # 查询指定的多个 IP 地址
    lip 199.91.73.222,178.79.131.110  # 查询指定的多个 IP 地址
    lip 199.91.73.222178.79.131.110  # 查询指定的多个 IP 地址
